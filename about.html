Tentang Website Ini

Website ini dibuat sebagai media pembelajaran dalam memahami dasar-dasar web development dan untuk latihan proses hosting web sederhana.

Tujuan Pembuatan

Website ini dirancang dengan tujuan-tujuan pembelajaran berikut:

Belajar membuat halaman web sederhana menggunakan HTML

Mengenal penggunaan hyperlink antar halaman

Mempelajari dasar styling dengan CSS dan Bootstrap

Melatih kemampuan upload website ke hosting

Teknologi yang Digunakan

Website ini dibangun menggunakan teknologi-teknologi modern:

HTML5
CSS3
JavaScript
Bootstrap 5
Google Fonts
Bootstrap Icons
